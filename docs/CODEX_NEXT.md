# Codex Next Tasks

## Immediate

1. Run `npm install`.
2. Create `.env` with DATABASE_URL and OLLAMA_URL.
3. Run `npm run db:push`.
4. Run `npm run dev`.
5. Open `http://localhost:3000`.
6. Click Seed Root Presence.
7. Visit `/chat` to copy a Presence ID.
8. Test POST `/api/chat` with that Presence ID.
